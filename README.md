## Getting Started

To get a local copy up and running follow these simple example steps.


### Prerequisites
- A browser
- A browser
- Node js (optional)

### Setup
1. Clone repository: 
```sh
  $ git clone https://github.com/SamroodAli/books-library.git

```
2. Change directory inside project
```sh
cd books-library
```
3. In your terminal, run 
```sh
npm i
```
